# Workshop with Vibe coding

## Structure
* Web => ReactJS
* API  => NodeJS and TypeScript
* Database => SQLite

## 1. Install Web with React + TS
* web
* React + TypeScript
```
$npm create vite@latest

$cd web
$npm install
$npm run dev
```

Run
```
$npm run dev
```

## 2. Install API
```
$mkdir api
$npm init -y
$npm install typescript tsx @types/node --save-dev
$npx tsc --init
```

Run 
```
$npx ts-node src/index.ts
```
